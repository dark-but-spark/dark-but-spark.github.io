Chapter 3 - Joint Distributions

随机向量

随机向量（Random Vector）是由多个随机变量组成的向量。设X1,X2,…,XnX_1, X_2, \ldots, X_nX1​,X2​,…,Xn​为nnn个随机变量，则随机向量X\mathbf{X}X定义为：

X=(X1,X2,…,Xn).\mathbf{X} = (X_1, X_2, \ldots, X_n). X=(X1​,X2​,…,Xn​).

如果ω∈Ω→(X1(ω),X2(ω),…,Xn(ω))∈Rn\omega \in \Omega \to (X_1(\omega), X_2(\omega), \ldots, X_n(\omega)) \in \mathbb{R}^nω∈Ω→(X1​(ω),X2​(ω),…,Xn​(ω))∈Rn，则称X\mathbf{X}X是一个nnn维随机向量。

联合分布

联合累计分布

随机向量X=(X1,X2,…,Xn)\mathbf{X} = (X_1, X_2, \ldots, X_n)X=(X1​,X2​,…,Xn​)的联合累计分布函数（Joint Cumulative Distribution Function, Joint CDF）定义为：

FX(x1,x2,…,xn)=P(X1≤x1,X2≤x2,…,Xn≤xn).F_{\mathbf{X}}(x_1, x_2, \ldots, x_n) = P(X_1 \leq x_1, X_2 \leq x_2, \ldots, X_n \leq x_n). FX​(x1​,x2​,…,xn​)=P(X1​≤x1​,X2​≤x2​,…,Xn​≤xn​).

接下来只讨论二维随机向量X=(X,Y)\mathbf{X} = (X, Y)X=(X,Y)的联合累计分布函数，记为FX,Y(x,y)F_{X,Y}(x,y)FX,Y​(x,y)。
满足以下性质：

  • 非减性：若x1<x2x_1 < x_2x1​<x2​且y1<y2y_1 < y_2y1​<y2​，则FX,Y(x1,y1)≤FX,Y(x2,y2)F_{X,Y}(x_1, y_1) \leq F_{X,Y}(x_2, y_2)FX,Y​(x1​,y1​)≤FX,Y​(x2​,y2​)；
  • 规范性：F(+∞，+∞)=1F(+\infty，+\infty)=1F(+∞，+∞)=1，F(−∞，y)=0F(-\infty，y)=0F(−∞，y)=0，F(x,−∞)=0F(x,-\infty)=0F(x,−∞)=0；
  • 对任意a<ba < ba<b且c<dc < dc<d，有

P(a<X≤b,c<Y≤d)=FX,Y(b,d)−FX,Y(a,d)−FX,Y(b,c)+FX,Y(a,c).P(a < X \leq b, c < Y \leq d) = F_{X,Y}(b,d) - F_{X,Y}(a,d) - F_{X,Y}(b,c) + F_{X,Y}(a,c). P(a<X≤b,c<Y≤d)=FX,Y​(b,d)−FX,Y​(a,d)−FX,Y​(b,c)+FX,Y​(a,c).

  • FX(x)=FX,Y(x,∞)F_X(x)=F_{X,Y}(x,\infty)FX​(x)=FX,Y​(x,∞)，FY(y)=FX,Y(∞,y)F_Y(y)= F_{X,Y}(\infty,y)FY​(y)=FX,Y​(∞,y)。

联合概率质量函数

对于离散型二维随机向量X=(X,Y)\mathbf{X} = (X, Y)X=(X,Y)，其联合概率质量函数（Joint Probability Mass Function, Joint PMF）定义为：

pX,Y(xi,yj)=P(X=xi,Y=yj).p_{X,Y}(x_i, y_j) = P(X = x_i, Y = y_j). pX,Y​(xi​,yj​)=P(X=xi​,Y=yj​).

满足以下性质：

  • 非负性：pX,Y(xi,yj)≥0p_{X,Y}(x_i, y_j) \geq 0pX,Y​(xi​,yj​)≥0；
  • 规范性：∑i∑jpX,Y(xi,yj)=1\sum_{i} \sum_{j} p_{X,Y}(x_i, y_j) = 1∑i​∑j​pX,Y​(xi​,yj​)=1；
  • 联合累计分布函数与联合概率质量函数的关系为：

FX,Y(x,y)=∑xi≤x∑yj≤ypX,Y(xi,yj).F_{X,Y}(x,y) = \sum_{x_i \leq x} \sum_{y_j \leq y} p_{X,Y}(x_i, y_j). FX,Y​(x,y)=xi​≤x∑​yj​≤y∑​pX,Y​(xi​,yj​).

  • PX(xi)=∑jPX,Y(xi,yj)P_{X}(x_i) = \sum_{j} P_{X,Y}(x_i, y_j)PX​(xi​)=∑j​PX,Y​(xi​,yj​)，PY(yj)=∑iPX,Y(xi,yj)P_{Y}(y_j) = \sum_{i} P_{X,Y}(x_i, y_j)PY​(yj​)=∑i​PX,Y​(xi​,yj​)。

联合概率密度函数

对于连续型二维随机向量X=(X,Y)\mathbf{X} = (X, Y)X=(X,Y)，其联合概率密度函数（Joint Probability Density Function, Joint PDF）定义为：

P(a≤X≤b,c≤Y≤d)=∫ab∫cdfX,Y(x,y) dy dx.P(a \leq X \leq b, c \leq Y \leq d) = \int_{a}^{b} \int_{c}^{d} f_{X,Y}(x,y) \, dy \, dx. P(a≤X≤b,c≤Y≤d)=∫ab​∫cd​fX,Y​(x,y)dydx.

满足以下性质：

  • 非负性：fX,Y(x,y)≥0f_{X,Y}(x,y) \geq 0fX,Y​(x,y)≥0；
  • 规范性：∬R2fX,Y(x,y) dy dx=1\iint_{\mathbb{R}^2} f_{X,Y}(x,y) \, dy \, dx = 1∬R2​fX,Y​(x,y)dydx=1；
  • 联合累计分布函数与联合概率密度函数的关系为：

FX,Y(x,y)=∫−∞x∫−∞yfX,Y(u,v) dv du.F_{X,Y}(x,y) = \int_{-\infty}^{x} \int_{-\infty}^{y} f_{X,Y}(u,v) \, dv \, du. FX,Y​(x,y)=∫−∞x​∫−∞y​fX,Y​(u,v)dvdu.

fX,Y(x,y)=∂2FX,Y(x,y)∂x∂y.f_{X,Y}(x,y) = \frac{\partial^2 F_{X,Y}(x,y)}{\partial x \partial y}. fX,Y​(x,y)=∂x∂y∂2FX,Y​(x,y)​.

  • fX(x)=∫−∞+∞fX,Y(x,y) dyf_X(x) = \int_{-\infty}^{+\infty} f_{X,Y}(x,y) \, dyfX​(x)=∫−∞+∞​fX,Y​(x,y)dy，fY(y)=∫−∞+∞fX,Y(x,y) dxf_Y(y) = \int_{-\infty}^{+\infty} f_{X,Y}(x,y) \, dxfY​(y)=∫−∞+∞​fX,Y​(x,y)dx。

条件分布

条件概率质量函数

对于离散型二维随机向量X=(X,Y)\mathbf{X} = (X, Y)X=(X,Y)，P(Y=yj)>0P(Y = y_j) > 0P(Y=yj​)>0时，XXX在给定Y=yjY = y_jY=yj​条件下的条件概率质量函数（Conditional Probability Mass Function, Conditional PMF）定义为：

pX∣Y(xi∣yj)=P(X=xi∣Y=yj)=P(X=xi,Y=yj)P(Y=yj)=pX,Y(xi,yj)pY(yj).p_{X|Y}(x_i | y_j) = P(X = x_i | Y = y_j) = \frac{P(X = x_i, Y = y_j)}{P(Y = y_j)} = \frac{p_{X,Y}(x_i, y_j)}{p_Y(y_j)}. pX∣Y​(xi​∣yj​)=P(X=xi​∣Y=yj​)=P(Y=yj​)P(X=xi​,Y=yj​)​=pY​(yj​)pX,Y​(xi​,yj​)​.

条件概率密度函数

对于连续型二维随机向量X=(X,Y)\mathbf{X} = (X, Y)X=(X,Y)，fY(y)>0f_Y(y) > 0fY​(y)>0时，XXX在给定Y=yY = yY=y条件下的条件概率密度函数（Conditional Probability Density Function, Conditional PDF）定义为：

fX∣Y(x∣y)=fX,Y(x,y)/fY(y).f_{X|Y}(x | y) = f_{X,Y}(x,y) / f_Y(y). fX∣Y​(x∣y)=fX,Y​(x,y)/fY​(y).

俩个随机变量之间的关系

独立性

随机向量X=(x1,x2,…,xn)\mathbf{X} = (x_1,x_2,\ldots,x_n)X=(x1​,x2​,…,xn​)的各个分量相互独立，当且仅当对于任意的x1,x2,…,xnx_1,x_2,\ldots,x_nx1​,x2​,…,xn​，有

FX1,X2,…,Xn(x1,x2,…,xn)=FX1(x1)FX2(x2)⋯FXn(xn).F_{X_1,X_2,\ldots,X_n}(x_1,x_2,\ldots,x_n) = F_{X_1}(x_1) F_{X_2}(x_2) \cdots F_{X_n}(x_n). FX1​,X2​,…,Xn​​(x1​,x2​,…,xn​)=FX1​​(x1​)FX2​​(x2​)⋯FXn​​(xn​).

对于离散随机变量，有

pX1,X2,…,Xn(x1,x2,…,xn)=pX1(x1)pX2(x2)⋯pXn(xn).p_{X_1,X_2,\ldots,X_n}(x_1,x_2,\ldots,x_n) = p_{X_1}(x_1) p_{X_2}(x_2) \cdots p_{X_n}(x_n). pX1​,X2​,…,Xn​​(x1​,x2​,…,xn​)=pX1​​(x1​)pX2​​(x2​)⋯pXn​​(xn​).

对于连续随机变量，有

fX1,X2,…,Xn(x1,x2,…,xn)=fX1(x1)fX2(x2)⋯fXn(xn).f_{X_1,X_2,\ldots,X_n}(x_1,x_2,\ldots,x_n) = f_{X_1}(x_1) f_{X_2}(x_2) \cdots f_{X_n}(x_n). fX1​,X2​,…,Xn​​(x1​,x2​,…,xn​)=fX1​​(x1​)fX2​​(x2​)⋯fXn​​(xn​).

性质

  • 若XXX和YYY相互独立，则E(g(X)h(Y))=E(g(X))E(h(Y))E(g(X)h(Y)) = E(g(X))E(h(Y))E(g(X)h(Y))=E(g(X))E(h(Y))。
  • 若XXX和YYY相互独立，则Var(g(X)+h(Y))=Var(g(X))+Var(h(Y))Var(g(X) + h(Y)) = Var(g(X)) + Var(h(Y))Var(g(X)+h(Y))=Var(g(X))+Var(h(Y))。

协方差

设随机变量XXX和YYY的期望存在，则称随机变量XXX和YYY的协方差（Covariance）为：

Cov(X,Y)=E[(X−E(X))(Y−E(Y))].Cov(X,Y) = E[(X - E(X))(Y - E(Y))]. Cov(X,Y)=E[(X−E(X))(Y−E(Y))].

  • 若Cov(X,Y)>0Cov(X,Y) > 0Cov(X,Y)>0，则称XXX和YYY正相关；
  • 若Cov(X,Y)<0Cov(X,Y) < 0Cov(X,Y)<0，则称XXX和YYY负相关；
  • 若Cov(X,Y)=0Cov(X,Y) = 0Cov(X,Y)=0，则称XXX和YYY不相关。
  • Cov(X,Y)=E(XY)−E(X)E(Y)=Var(X+Y)−Var(X)−Var(Y)2Cov(X,Y) = E(XY) - E(X)E(Y)=\frac{Var(X+Y)-Var(X)-Var(Y)}{2}Cov(X,Y)=E(XY)−E(X)E(Y)=2Var(X+Y)−Var(X)−Var(Y)​。
  • Covariance-Variance relationship: Cov(X,X)=Var(X)Cov(X,X) = Var(X)Cov(X,X)=Var(X), Var(X±Y)=Var(X)+Var(Y)±2Cov(X,Y)Var(X\pm Y) = Var(X) + Var(Y) \pm 2Cov(X,Y)Var(X±Y)=Var(X)+Var(Y)±2Cov(X,Y)。
  • Symmetry: Cov(X,Y)=Cov(Y,X)Cov(X,Y) = Cov(Y,X)Cov(X,Y)=Cov(Y,X).
  • Constants: For any constant aaa and bbb, Cov(aX,bY)=abCov(X,Y)Cov(aX,bY) = abCov(X,Y)Cov(aX,bY)=abCov(X,Y).
  • Linearity: For any random variables XXX, YYY, and ZZZ, Cov(X+Y,Z)=Cov(X,Z)+Cov(Y,Z)Cov(X+Y,Z) = Cov(X,Z) + Cov(Y,Z)Cov(X+Y,Z)=Cov(X,Z)+Cov(Y,Z).
  • Constants have zero covariance: For any constant ccc, Cov(X,c)=0Cov(X,c) = 0Cov(X,c)=0.
  • Bilinearity: For any random variables X1X_1X1​, X2X_2X2​, Y1Y_1Y1​, and Y2Y_2Y2​, and constants aaa, bbb, ccc, and ddd, Cov(a1X1+a2X2+⋯+anXn,b1Y1+b2Y2+⋯+bmYm)=∑i=1n∑j=1maibjCov(Xi,Yj)Cov(a_1X_1 + a_2X_2 + \cdots+a_n X_n, b_1Y_1 + b_2Y_2 + \cdots+b_m Y_m) = \sum_{i=1}^n \sum_{j=1}^m a_i b_j Cov(X_i, Y_j)Cov(a1​X1​+a2​X2​+⋯+an​Xn​,b1​Y1​+b2​Y2​+⋯+bm​Ym​)=∑i=1n​∑j=1m​ai​bj​Cov(Xi​,Yj​).

相关系数

设随机变量XXX和YYY的方差均存在且不为零，则称随机变量XXX和YYY的相关系数（Correlation Coefficient）为：

ρX,Y=Cor(X,Y)=E[(X−E(X))(Y−E(Y))SD(X)⋅SD(Y)]=Cov(X,Y)Var(X)Var(Y).\rho_{X,Y} = Cor(X,Y) = E\left[\frac{(X-E(X))(Y-E(Y))}{SD(X) \cdot SD(Y)}\right]=\frac{Cov(X,Y)}{\sqrt{Var(X)Var(Y)}}. ρX,Y​=Cor(X,Y)=E[SD(X)⋅SD(Y)(X−E(X))(Y−E(Y))​]=Var(X)Var(Y)​Cov(X,Y)​.

  • 相关系数的取值范围为−1≤ρX,Y≤1-1 \leq \rho_{X,Y} \leq 1−1≤ρX,Y​≤1。
  • 若ρX,Y=1\rho_{X,Y} = 1ρX,Y​=1，则称XXX和YYY完全正相关；
  • 若ρX,Y=−1\rho_{X,Y} = -1ρX,Y​=−1，则称XXX和YYY完全负相关；
  • 若ρX,Y=0\rho_{X,Y} = 0ρX,Y​=0，则称XXX和YYY不相关。
  • ρX,Y\rho_{X,Y}ρX,Y​描绘了变量间线性关系的强度和方向，但不一定表示因果关系。

均方误差

假设存在一个线性估计器Y^=aX+b\hat{Y} = aX + bY^=aX+b来预测随机变量YYY的值，其中aaa和bbb是常数。均方误差（Mean Squared Error, MSE）定义为：

MSE=E[(Y−Y^)2]=E[(Y−(aX+b))2].MSE = E\left[(Y - \hat{Y})^2\right] = E\left[(Y - (aX + b))^2\right]. MSE=E[(Y−Y^)2]=E[(Y−(aX+b))2].

  • 最优线性估计器的系数aaa和bbb可以通过最小化均方误差来确定，具体为：

{a=Cov(X,Y)Var(X),b=E(Y)−aE(X).\begin{cases} a = \frac{Cov(X,Y)}{Var(X)},\\b = E(Y) - aE(X). \end{cases} {a=Var(X)Cov(X,Y)​,b=E(Y)−aE(X).​

  • 最小均方误差（Minimum Mean Squared Error, MMSE）为：

min⁡a,bMSE=Var(Y)−[Cov(X,Y)]2Var(X)=Var(Y)(1−ρX,Y2).\min_{a,b} MSE = Var(Y) - \frac{[Cov(X,Y)]^2}{Var(X)}=Var(Y)(1-\rho_{X,Y}^2). a,bmin​MSE=Var(Y)−Var(X)[Cov(X,Y)]2​=Var(Y)(1−ρX,Y2​).

条件期望

设随机变量XXX和YYY的联合分布已知，则在给定Y=yY=yY=y条件下，随机变量XXX的条件期望（Conditional Expectation）定义为：

  • 对于离散型随机变量，有

E(X∣Y=y)=∑ixiP(X=xi∣Y=y)=∑ixiP(X=xi,Y=y)P(Y=y).E(X|Y=y) = \sum_{i} x_i P(X=x_i | Y=y) = \sum_{i} x_i \frac{P(X=x_i, Y=y)}{P(Y=y)}. E(X∣Y=y)=i∑​xi​P(X=xi​∣Y=y)=i∑​xi​P(Y=y)P(X=xi​,Y=y)​.

  • 对于连续型随机变量，有

E(X∣Y=y)=∫−∞+∞xfX∣Y(x∣y)dx=∫−∞+∞xfX,Y(x,y)fY(y)dx.E(X|Y=y) = \int_{-\infty}^{+\infty} x f_{X|Y}(x|y) dx = \int_{-\infty}^{+\infty} x \frac{f_{X,Y}(x,y)}{f_Y(y)} dx. E(X∣Y=y)=∫−∞+∞​xfX∣Y​(x∣y)dx=∫−∞+∞​xfY​(y)fX,Y​(x,y)​dx.

全期望公式

随机变量XXX的期望可以通过条件期望来计算，称为全期望公式（Law of Total Expectation）：

E(x)=E[E(X∣Y)].E(x) = E[E(X|Y)]. E(x)=E[E(X∣Y)].

具体地，

E(X)=E[E(X∣Y)]=∑jE(X∣Y=yj)P(Y=yj)（对于离散型随机变量）E(X) = E[E(X|Y)] = \sum_{j} E(X|Y=y_j) P(Y=y_j) \text{（对于离散型随机变量）} E(X)=E[E(X∣Y)]=j∑​E(X∣Y=yj​)P(Y=yj​)（对于离散型随机变量）

E(X)=E[E(X∣Y)]=∫−∞+∞E(X∣Y=y)fY(y)dy（对于连续型随机变量）E(X) = E[E(X|Y)] = \int_{-\infty}^{+\infty} E(X|Y=y) f_Y(y) dy \text{（对于连续型随机变量）} E(X)=E[E(X∣Y)]=∫−∞+∞​E(X∣Y=y)fY​(y)dy（对于连续型随机变量）
